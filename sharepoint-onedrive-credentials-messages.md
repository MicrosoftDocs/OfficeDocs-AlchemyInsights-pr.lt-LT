# <a name="credential-messages-in-sharepoint-or-onedrive"></a>Kredencialų laiškus SharePoint arba "OneDrive"

<p>Vartotojai gali būti periodiškai raginimus tarnybos įvesti kredencialus SharePoint Online ir "OneDrive" bandant atidaryti dokumentus.</p> <p>Toliau pateikiami kai kurie pavyzdžiai kredencialų pranešimų:</p> <p>&bull;<span style="mso-tab-count: 1;"> &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; </span>Prisijungti prie organizacijos abonemento</p> <p>&bull;<span style="mso-tab-count: 1;"> &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; </span>Kredencialai yra reikalaujama</p> <p>&bull;<span style="mso-tab-count: 1;"> &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; </span>Mes negalime prisijungti dabar, patikrinkite tinklo ryšį.</p> <p>Jei jūs patiriate šių simptomų, prašome pabandyti už <a href="https://support.microsoft.com/en-us/help/2913639/office-applications-periodically-prompt-for-credentials-to-sharepoint">šiame</a>straipsnyje nurodytus veiksmus.</p> <p>Kita vertus, jei naudojate bendrosios autentifikacijos (SSO) autentifikuoti, prašome kreiptis į <a href="https://support.microsoft.com/en-us/help/4025962/cant-sign-in-after-update-to-office-2016-build-16-0-7967-on-windows-10">šį straipsnį</a> vietoj.</p> <p>&nbsp;</p>

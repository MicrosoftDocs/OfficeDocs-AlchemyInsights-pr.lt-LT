# <a name="irm-protection-to-sharepoint-files"></a>IRM apsauga į SharePoint failai


<p>Per SharePoint Online, IRM apsauga taikoma failų sąrašą ir bibliotekos lygmenyje. Organizacijos gali naudoti IRM apsaugą, pirmiausia turite nustatyti teisių valdymo. IRM remiasi Azure teisių valdymo tarnybos, Azure duomenų apsaugos užšifruoti ir priskirti naudojimo apribojimai. Kai kurie "Office 365" planus įtraukti Azure teisių valdymo, bet ne visus. Norėdami sužinoti daugiau, skaitykite <a href="https://docs.microsoft.com/azure/information-protection/understand-explore/office-apps-services-support" data-linktype="external">kaip Office taikomųjų programų ir paslaugų palaikymo Azure teisių valdymą</a>.</p> <ul> <li><a href="https://docs.microsoft.com/en-us/office365/securitycompliance/set-up-irm-in-sp-admin-center">Nustatyti iki informacijos teisių valdymo (IRM) SharePoint administravimo centro</a></li> <li><a href="https://docs.microsoft.com/en-us/office365/securitycompliance/set-up-irm-in-sp-admin-center#irm-enable-sharepoint-document-libraries-and-lists">IRM įgalinti SharePoint dokumentų bibliotekose ir sąrašuose</a></li> <li><a href="https://support.office.com/en-US/Article/Information-Rights-Management-in-Office-c7a70797-6b1e-493f-acf7-92a39b85e30c">Informacijos teisių valdymas programų pakete „Office“</a></li> <li><a href="https://docs.microsoft.com/en-us/office365/SecurityCompliance/information-rights-management-in-exchange-online">Informacijos teisių valdymo biržoje internetu</a></li> </ul>

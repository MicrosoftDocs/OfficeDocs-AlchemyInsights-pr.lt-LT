# <a name="download-and-share-updates"></a>Naujinimų atsisiuntimas ir bendrinimas

Užuot parsisiuntę naujinimus atskirai, pristatymo optimizavimo atsisiuntimo režimas įgalina bendrinimą keliuose kompiuteriuose. Daugiau informacijos ieškokite [Delivery Optimization for Windows 10 updates](https://docs.microsoft.com/windows/deployment/update/waas-delivery-optimization).  

Galimi atsisiuntimo režimai:  
- Tik HTTP  
- Lan  
- Grupės  
- Interneto  
- Paprasta  
- Apeiti

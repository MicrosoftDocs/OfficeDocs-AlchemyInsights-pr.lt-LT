# <a name="change-the-font-size-in-a-message"></a>Šrifto dydžio keitimas laiške

1. Jei jau įvedėte laišką, pažymėkite tekstą.
2. Pažymėkite po pranešimu esančią piktogramą  **formatas** (tai pirmoji piktograma).
3. Pasirinkite piktogramą  **šrifto dydis**  virš pranešimo (tai Septintoji piktograma).
4. Pasirinkite dydį.

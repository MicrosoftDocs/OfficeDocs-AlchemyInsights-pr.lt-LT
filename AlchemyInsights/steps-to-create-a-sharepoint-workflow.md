# <a name="steps-to-create-a-sharepoint-workflow"></a>"SharePoint" darbo eigos kūrimo veiksmai

- Jei negalite įtraukti SharePoint 2010 patvirtinimo darbo eigos, [žr.: Negalima įtraukti numatytąją 2010 m. patvirtinimo darbo eigą](https://docs.microsoft.com/alchemyinsights/can-t-add-default-2010-approval-workflow).
- Kaip sukurti darbo eigas:
    - [2013 m. darbo eiga iš SharePoint Designer](https://docs.microsoft.com/sharepoint/dev/general-development/creating-a-workflow-by-using-sharepoint-designer-and-the-sharepoint-wo)
    - [2010 darbo eiga iš SharePoint Designer](https://support.office.com/article/introduction-to-designing-and-customizing-workflows-32c9c0bf-5e20-4f74-8b9c-d3ea79f2962b)

- Darbo eigos trikčių diagnostika ir derinimas:
    - Derinimo darbo eigos panašus į ankstesnių versijų.  Daugiau informacijos ieškokite [SharePoint darbo eigų derinimas](https://docs.microsoft.com/sharepoint/dev/general-development/debugging-sharepoint-server-workflows).
    - Daugiau informacijos apie trikčių diagnostiką ieškokite [SharePoint Server darbo eigos tikrinimo klaidų šalinimas](https://docs.microsoft.com/sharepoint/dev/general-development/troubleshooting-sharepoint-server-workflow-validation-errors-in-visio).

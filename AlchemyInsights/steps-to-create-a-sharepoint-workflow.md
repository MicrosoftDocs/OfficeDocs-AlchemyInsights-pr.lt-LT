# <a name="steps-to-create-a-sharepoint-workflow"></a>"SharePoint" darbo eigos kūrimo veiksmai

- Jei negalite pridėti SharePoint 2010 patvirtinimo darbo eigos, žiūrėkite [negalima pridėti numatytojo 2010 patvirtinimo darbo eigos](https://docs.microsoft.com/alchemyinsights/can-t-add-default-2010-approval-workflow).
- Kaip sukurti darbo eigas:
    - [2013 darbo eiga iš SharePoint Designer](https://docs.microsoft.com/sharepoint/dev/general-development/creating-a-workflow-by-using-sharepoint-designer-and-the-sharepoint-wo)
    - [2010 darbo eiga iš SharePoint Designer](https://support.office.com/article/introduction-to-designing-and-customizing-workflows-32c9c0bf-5e20-4f74-8b9c-d3ea79f2962b)
    - [Įvadas į "SharePoint" darbo eigą](https://support.office.com/article/introduction-to-sharepoint-workflow-07982276-54e8-4e17-8699-5056eff4d9e3)

- Darbo eiga trikčių diagnostika ir derinimas:
    - Derinimo darbo eigos primena ankstesnes versijas.  Daugiau informacijos ieškokite [SharePoint darbo eigų derinimas](https://docs.microsoft.com/sharepoint/dev/general-development/debugging-sharepoint-server-workflows).
    - Daugiau informacijos apie trikčių šalinimą ieškokite [SharePoint Server darbo eigos tikrinimo klaidų šalinimas](https://docs.microsoft.com/sharepoint/dev/general-development/troubleshooting-sharepoint-server-workflow-validation-errors-in-visio).

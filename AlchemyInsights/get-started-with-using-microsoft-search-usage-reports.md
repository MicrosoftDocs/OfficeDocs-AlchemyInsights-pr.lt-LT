# <a name="get-started-with-using-microsoft-search-usage-reports"></a>Darbo su "Microsoft Search" naudojimo ataskaitomis pradžia

"Microsoft Search" naudojimo ataskaitose pateikiama įžvalgų, kaip ieška veikia jūsų organizacijoje. Šios įžvalgos padės jums rasti turinį ir padaryti iešką naudingesnėmis ir puikesnėmis jūsų organizacijos vartotojams.

Norėdami sužinoti daugiau, žr. ["Microsoft" ieškos naudojimo ataskaitos](https://go.microsoft.com/fwlink/?linkid=2152048).

# <a name="removing-the-last-on-premises-exchange-server"></a>Paskutinio vietinio "Exchange" serverio šalinimas

Prieš pašalindami paskutinį "Exchange" serverį iš savo vietinėje organizacijoje, turite įsitikinti, kad suprantate, kaip tai padaryti, kad galėtumėte nuspręsti, ar jums reikia paskutinio serverio kitais tikslais. Norėdami gauti daugiau informacijos, Sužinokite, [kaip ir kada panaikinti savo vietinius "Exchange" serverius hibridinėje įdiegtyje](https://technet.microsoft.com/library/dn931280%28v=exchg.150%29.aspx).

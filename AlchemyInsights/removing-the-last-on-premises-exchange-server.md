# <a name="removing-the-last-on-premises-exchange-server"></a>Paskutinio vietinio "Exchange" serverio šalinimas

Prieš pašalindami paskutinį "Exchange" serverį iš vietinės organizacijos, įsitikinkite, kad suprantate, kokių pasekmių turite padaryti, kad galėtumėte nuspręsti, ar jums reikia paskutinio serverio kitais tikslais. Jei norite gauti daugiau informacijos, patikrinkite [Kaip ir kada nutraukti savo vietiniame Exchange serverių hibridinis diegimas](https://technet.microsoft.com/library/dn931280%28v=exchg.150%29.aspx).

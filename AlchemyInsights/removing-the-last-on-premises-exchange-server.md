# <a name="removing-the-last-on-premises-exchange-server"></a>Pašalinti paskutinę vietiniame Exchange serverio

Prieš išimdami paskutiniais Exchange serverio iš jūsų vietinėje organizacijoje, turėtumėte įsitikinti, kad jūs suprantate, kokia tai padaryti, siekiant padėti jums nuspręsti, jei reikia, paskutinio serveriui jokiems kitiems tikslams. Norėdami gauti daugiau informacijos, patikrinti [kaip ir kada uždaryti jūsų vietiniame Exchange serverio hibridinę įdiegtį](https://technet.microsoft.com/en-us/library/dn931280%28v=exchg.150%29.aspx).

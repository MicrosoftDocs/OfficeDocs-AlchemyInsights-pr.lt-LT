# <a name="limit-sharing-in-microsoft-365"></a>"Microsoft 365" bendro naudojimo apribojimas

Nors negalite visiškai išjungti vidinio bendrinimo arba pašalinti bendrinimo mygtuko iš svetainių, yra įvairių būdų, kaip galite apriboti bendrinimą "Microsoft 365", kad būtų patenkinti jūsų organizacijos poreikiai. Daugiau informacijos rasite ["Microsoft 365" bendro naudojimo apribojimas](https://docs.microsoft.com/Office365/Enterprise/microsoft-365-limit-sharing).

# <a name="domain-status---no-services-selected"></a>Domeno būsena – neparinktos jokios tarnybos

Domeno sveikatos būsena pasiekiama prijungus domeną tiesiogiai prie Microsoft 365, pvz., Exchange. Norėdami tai padaryti, eikite į Microsoft 365 administravimo centrą ir pasirinkite **DNS įrašai Dns**  >  **valdymas**.

Jei negalite tiesiogiai prijungti domeno prie Microsoft 365 tarnybų, domeno sveikatos būsena negalima. Pvz., kai naudojate pasirinktinę DNS kelvadą el. paštui, išorinio pašto šiukšlių filtravimo tarnyboms arba naudojate Exchange vietines ir Exchange Online (dar vadinamas hibridine įdiegtimi).

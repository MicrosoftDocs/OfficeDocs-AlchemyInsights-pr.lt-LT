# <a name="convert-a-user-mailbox-to-a-shared-mailbox"></a>Konvertuoti vartotojo pašto dėžutės į bendrinamą pašto dėžutę

Konvertuojant į bendrai naudojamos pašto dėžutės vartotojo pašto dėžutės, yra išsaugotos visos esamos el. pašto ir kalendoriaus. Tik dabar jis yra bendrai naudojamos pašto dėžutės, kur keli žmonės galės jį pasiekti vietoj vieno asmens. Vėliau, galite konvertuoti bendrinamos pašto dėžutės atgal į vartotojo pašto dėžutės (privataus).

# <a name="create-users-in-hybrid-deployments"></a>Kurdami vartotojus hibridinės įdiegtys

Kai jūsų organizacija yra sambūvio tarp vietinės aplinkos ir "Office 365", jums reikia sukurti visų vartotojo objektų vietinės Active Directory. Galite sukurti vartotojo objektams tinkamas organizacijos vieneto (OU), jie yra sinchronizuojami į jūsų "Office 365" organizacijai.

# <a name="create-users-in-hybrid-deployments"></a>Vartotojų kūrimas hibridiniuose diegimuose

Kai jūsų organizacija yra hibridinis diegimas tarp vietinės aplinkos ir "Microsoft 365", turite sukurti visus vartotojo objektus vietinėje "Active Directory". Galite sukurti vartotojo objektų atitinkamame organizacijos vienete (OU), kad jie sinchronizuojami su jūsų organizacija.

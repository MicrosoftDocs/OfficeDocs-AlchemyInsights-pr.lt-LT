# <a name="distribution-group-not-showing-in-groups-list"></a>Siuntimo grupė nerodoma grupių sąraše

Gali užtrukti iki 60 minučių, kol jūsų naujas siuntimo sąrašas bus rodomas administravimo centre.
  
Jei norite įtraukti narių arba atlikti kitus pakeitimus dabar, naudokite funkciją grupės " [Exchange" administravimo centre](https://outlook.office365.com/ecp/?rfr=Admin_o365&amp;exsvurl=1).

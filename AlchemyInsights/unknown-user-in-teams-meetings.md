# <a name="unknown-user-appears-in-teams-meetings-or-chats"></a>"Nežinomas vartotojas" pasirodo komandų susitikimuose arba pokalbiuose

Vartotojas gali būti pašalintas iš "Microsoft 365". Šiuo atveju pabandykite panaikinti vartotoją iš "teams".  

Pabandykite išvalyti šią talpyklą, kai komandos yra uždarytos, tada iš naujo paleiskite komandas: `C:\Users\<USER>\AppData\Roaming\Microsoft\Teams`

Norėdami gauti naujinimų šiuo klausimu, žiūrėkite  [pašalinti vartotoją programoje "Microsoft teams" rodomas kaip "nežinomas vartotojas"](https://docs.microsoft.com/MicrosoftTeams/troubleshoot/known-issues/removed-user-appears-as-unknown).

# <a name="hybrid-configuration-wizard"></a>Hibridinės konfigūracijos vedlys

Hibridinės konfigūracijos vedlys yra programėlė, padedanti nustatyti naują hibridinį diegimą tarp vietinės "Exchange" organizacijos ir "Microsoft 365". Jei jau esate nustatę hibridinį diegimą, vedlys padės atnaujinti hibridinio diegimo parametrus. Norėdami gauti daugiau informacijos, patikrinkite:
  
- [Hibridinio konfigūravimo vedlio paleidimas](https://technet.microsoft.com/library/mt595788%28v=exchg.150%29.aspx)

- [Hibridinės konfigūracijos vedlys](https://technet.microsoft.com/library/hh529921%28v=exchg.150%29.aspx)

- [Duk apie hibridinės konfigūracijos vedlį](https://technet.microsoft.com/library/mt488940%28v=exchg.150%29.aspx)

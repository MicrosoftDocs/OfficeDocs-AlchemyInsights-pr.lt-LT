# <a name="windows-doesnt-start-on-surface"></a>"Windows" nepaleidžiama paviršiuje

Jei "Windows" nepaleidžiama jūsų paviršiuje arba įrenginys nepažadina miego režimo, išbandykite šiuos trikčių šalinimo veiksmus:

- Jei "Windows" nepavyksta tinkamai paleisti, kai įjungiate savo paviršių (pvz., "Windows" nepaleidžiama), veiksmų, kad išspręstumėte problemą, ieškokite ["Windows" nepaleidžiama paviršiuje](https://support.microsoft.com/surface/windows-doesn-t-start-on-surface-3dd47ea1-472a-4514-c8e1-ff81bd72be5c).

- Jei jūsų paviršius nebus įjungtas arba pažadinamas, trikčių šalinimo veiksmai Peržiūrėkite, kad nebūtų [įjungiamas arba paleidžiamas iš miego režimo](https://support.microsoft.com/surface/surface-won-t-turn-on-or-wake-from-sleep-1e181652-3db8-5ca1-9649-7390fafb102a).

- Jei naudojate trečiosios šalies įrenginį, peržiūrėkite [trečiosios šalies įrenginių trikčių diagnostiką](https://support.microsoft.com/topic/b6f3408d-dac9-43e2-82f6-e620ca783636).

# <a name="pst-data-files-in-outlookcom"></a>PST duomenų failai "Outlook.com"

Negalima importuoti "Outlook". PST duomenų failą į savo "Outlook.com" paskyrą naudodami naršyklę.

Norėdami importuoti. PST duomenų failą, turite naudoti "Outlook" kompiuterio taikomąją programą. Norėdami sužinoti daugiau, skaitykite [importavimo el. pašto, kontaktų ir kalendoriaus importavimas iš "Outlook. pst".](https://support.office.com/article/431a8e9a-f99f-4d5f-ae48-ded54b3440ac?wt.mc_id=Office_Outlook_com_Alchemy)

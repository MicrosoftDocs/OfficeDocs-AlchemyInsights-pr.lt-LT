# <a name="public-folder-access-from-mobile-devices"></a>Viešojo aplanko prieiga iš mobiliųjų įrenginių

Viešieji aplankai nėra prieinama iš mobiliųjų įrenginių.
  
Jei norite, kad funkcijos panašios į viešąjį aplanką ir jį pasiekti iš mobiliųjų įrenginių, [sužinoti apie Office 365 grupes](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) ieškokite geresnė alternatyva.

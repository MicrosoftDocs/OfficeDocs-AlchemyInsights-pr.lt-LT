# <a name="restore-the-sharepoint-root-site"></a>"SharePoint" šaknies svetainės atkūrimas

Atkurkite panaikintą šaknies svetainę iš naujo "SharePoint" administravimo centro. Kai atkuriama, vietoj to pakeiskite svetainę.

1. Eikite į puslapį [panaikintos svetainės](https://admin.microsoft.com/sharepoint?page=recycleBin&modern=true) . 
2. Pasirinkite atkurtą šaknies svetainę.
3. Spustelėkite **atkurti**.
4. [Pakeiskite savo šaknies svetainę](https://docs.microsoft.com/sharepoint/troubleshoot/sites/url-that-resides-under-root-site-collection-is-broken).

Daugiau informacijos rasite [modernizuoti savo šaknies svetainę](https://docs.microsoft.com/sharepoint/modern-root-site).

# <a name="control-sync-throughput"></a>Sinchronizavimo pralaidumo valdymas

Jei reikia valdyti sinchronizavimo programų srautą, rekomenduojame naudoti tinklo paslaugų kokybės (QoS) strategijas arba "Windows QoS" strategijas, jei įmanoma. Informacijos apie tinklo parametrus, kuriuos galite padaryti prieinamus vartotojams, žr. ["OneDrive" sinchronizavimo taikomosios programos nusiuntimo arba atsisiuntimo greičio keitimas.](https://support.office.com/article/71cc69da-2371-4981-8cc8-b4558bdda56e)

# <a name="copy-and-paste-in-outlookcom"></a>Kopijavimas ir įklijavimas Outlook.com

Outlook.com nėra kopijavimo arba įklijavimo mygtukų, tačiau galite naudoti sparčiuosius klavišus:

**Kompiuteryje:** CTRL+C kopijuoti, CTRL +X iškirpti, ir CTRL +V įklijuoti.

**"Mac":** CMD + C kopijuoti, CMD + X iškirpti, ir CMD + V įklijuoti.

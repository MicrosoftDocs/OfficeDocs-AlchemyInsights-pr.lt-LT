# <a name="copy-and-paste-in-outlookcom"></a>Nukopijuokite ir įklijuokite į "Outlook.com"

Yra ne kopijuoti arba įklijuoti mygtukus programoje "Outlook.com", bet jūs galite naudoti sparčiuosius klavišus:

**PC:** CTRL + C Norėdami kopijuoti, iškirpti CTRL + X ir CTRL + V, kad įklijuotumėte.

**"Mac":** CMD + C, kad nukopijuotumėte, CMD + X – iškirpti ir CMD + V, kad įklijuotumėte.

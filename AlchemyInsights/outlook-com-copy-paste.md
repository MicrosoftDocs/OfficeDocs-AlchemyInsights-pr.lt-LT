# <a name="copy-and-paste-in-outlookcom"></a>Kopijavimas ir įklijavimas naudojant "Outlook.com"

Nėra mygtukų kopijuoti arba įklijuoti "Outlook.com", bet galite naudoti sparčiuosius klavišus:

**Asmeniniame kompiuteryje:** CTRL + C, kad nukopijuotumėte, CTRL + X, kad iškirptumėte, ir CTRL + V, kad įklijuotumėte.

**"Mac" kompiuteryje:** CMD + C, kad nukopijuotumėte, CMD + X, ir CMD + V, kad įklijuotumėte.

# <a name="onedrive-troubleshoot-processing-changes"></a>"OneDrive": "apdorojimo keitimo" trikčių diagnostika

Jei ilgesnį laiką matote pranešimą "apdorojimo keitimas", gali būti, kad atidarėte internetinį failą, užstrigęs eilėje failus arba labai didelį failų sinchronizavimą, skaitykite " [OneDrive": trikčių šalinimo keitimas](https://support.office.com/article/onedrive-is-stuck-on-processing-changes-b386b813-9b66-4e47-8c4c-2b45533edccd) , kad būtų lengviau išspręsti šią problemą.

Taip pat matysite šį pranešimą, jei turite failą, kuris pakeis visą laiką (pvz., "Outlook". PST failas arba duomenų bazė). Sužinokite, kaip [perkelti "Outlook" duomenų failus](https://support.office.com/article/how-to-remove-an-outlook-pst-data-file-from-onedrive-b6b9e522-59bd-40f7-949f-168d0aa9b38e) , kuriuos reguliariai naudojate iš "OneDrive".

# <a name="how-to-download-and-install-microsoft-to-do"></a>Kaip atsisiųsti ir įdiegti "Microsoft to do"

"Microsoft", skirtą "Win32", "iOS", "Android" ir "Mac" programėlei, galima atsisiųsti iš tinkamos saugyklos.

Jei turite savo nuomotojo ribotą saugyklos prieigą, skaitykite [https://docs.microsoft.com/microsoft-store/distribute-offline-apps](https://docs.microsoft.com/en-us/microsoft-store/distribute-offline-apps) apie diegimo neprisijungus programą

Jei norite atsisiųsti taikomąją programą iš parduotuvės naudodami **darbo** paskyrą, ji turi būti publikuojama jūsų įmonės **asmeninime parduotuvėje.** Daugiau informacijos rasite [https://docs.microsoft.com/en-us/microsoft-store/app-inventory-management-microsoft-store-for-business#private-store-availability](https://docs.microsoft.com/microsoft-store/app-inventory-management-microsoft-store-for-business#private-store-availability)

# <a name="hub-site-permissions"></a>Koncentratoriaus svetainės teisės

Koncentratoriaus svetainės teisių valdymas priklauso nuo esamo svetainės tipo. Jei svetainė yra prie grupės prijungta komandos svetainė, turite valdyti teises naudodami "Microsoft" 365 grupę. Jei tai bendravimo svetainė, turite valdyti teises naudodami "SharePoint" grupes.

- [Koncentratoriaus svetainės teisės](https://docs.microsoft.com/sharepoint/modern-experience-sharing-permissions#hub-site-permissions)  
- ["Hub" svetainių planavimas](https://docs.microsoft.com/sharepoint/planning-hub-sites)

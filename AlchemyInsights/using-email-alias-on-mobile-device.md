# <a name="email-aliases-on-mobile-devices"></a>El. pašto pseudonimai mobiliuosiuose įrenginiuose

El. laiško pseudonimas – tai dar vienas pašto adresas, sukonfigūruotas pašto dėžutėje, todėl laiškai, siunčiami vartotojo pseudonimams, bus pristatyti į jo pašto dėžutę ir bus perskaityti jų mobiliajame įrenginyje. Nereikia jokių papildomų konfigūracijų.

Tačiau vartotojas negali atsakyti į laišką arba siųsti naujų žinučių naudodami pseudonimą kaip adresą. Visi Išeinantys laiškai bus iš jų pirminio elektroninio pašto adreso.

# <a name="port-google-chrome-extensions-to-microsoft-edge-chromium"></a>"Google Chrome" plėtiniai "Microsoft Edge" (chromas)

" [Google Chrome" plėtinius paprasta perkelti į "Microsoft Edge" (chromo)](https://docs.microsoft.com/microsoft-edge/extensions-chromium/developer-guide/port-chrome-extension). Daugeliu atvejų norint paleisti šiuos plėtinius "Microsoft Edge", reikia tik minimalių pasikeitimų.

"Google Chrome" palaikomi plėtinių API ir MANIFEST yra suderinami su "Microsoft Edge". Tačiau "Microsoft Edge" nepalaiko plėtinio API Chrome. GCM, Chrome. Identity. getAccounts, Chrome. Identity. getAuthToken ir Chrome. instanceID.

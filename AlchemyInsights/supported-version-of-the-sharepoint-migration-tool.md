# <a name="supported-version-of-the-sharepoint-migration-tool"></a>Palaikoma versija SharePoint migracijos įrankis

Šiuo SharePoint migracijos įrankis palaiko tik migracijų iš SharePoint 2013. Šiuo metu nepalaiko versijos 2010 m. ir 2016 m.
  
Norėdami gauti daugiau informacijos, peržiūrėkite [įvesti perkėlimo įrankis SharePoint](https://go.microsoft.com/fwlink/?linkid=2044765&amp;clcid=0x409).

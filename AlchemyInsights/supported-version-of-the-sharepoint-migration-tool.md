# <a name="supported-version-of-the-sharepoint-migration-tool"></a>Palaikoma "SharePoint" perkėlimo įrankio versija



Šiuo metu SharePoint perkėlimo įrankis palaiko tik perkėlimus iš SharePoint 2013. Šiuo metu nepalaikomos versijos 2010 ir 2016.
  
Daugiau informacijos ieškokite " [SharePoint" perkėlimo įrankio pristatymas](https://go.microsoft.com/fwlink/?linkid=2044765&amp;clcid=0x409).

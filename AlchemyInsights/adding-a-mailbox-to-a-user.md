# <a name="adding-a-mailbox-to-a-user"></a>Pašto dėžutės įtraukimas į vartotoją

Norint įtraukti pašto dėžutę į jau sukurtą vartotoją arba naują vartotoją, vartotojui turi būti priskirta "Exchange Online" licencija. Jei vartotojas jau turi priskirtas licencijas, pasirinkite Exchange Online licencijos puslapio skyriuje Programėlės, kad pradėtumėte procesą, kad sukurtumėte vartotojo pašto dėžutę. Daugiau informacijos rasite [Licencijų priskyrimas vartotojams](https://docs.microsoft.com/microsoft-365/admin/manage/assign-licenses-to-users).

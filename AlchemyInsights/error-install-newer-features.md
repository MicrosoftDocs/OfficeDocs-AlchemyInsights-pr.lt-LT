# <a name="error-your-new-install-has-newer-features-than-your-current-version-do-you-want-to-activate"></a>Klaida: jūsų naujojoje įdiegtyje yra naujesnės funkcijos nei dabartinėje versijoje. Ar norite aktyvinti...

Norėdami išspręsti šią "Office" klaidą, pasirinkite savo "Office" versijai automatinį sprendimą:

- Jei turite "Microsoft 365", atsisiųskite ir paleiskite [palaikymo ir atkūrimo pagalbinę priemonę](https://aka.ms/SaRA-OfficeActivation-Chat). Vykdykite nurodymus ir raginimus, kad išspręstumėte "Office" aktyvinimo klaidą.

- Jei turite "Office 2019" arba "Office 2016", atlikite veiksmus, pateiktus [nelicencijuotų produktų ir aktyvinimo klaidų "Office](https://support.microsoft.com/office/0d23d3c0-c19c-4b2f-9845-5344fedc4380#bkmk_fixyourself)".

- Jei turite "Office 2013", peržiūrėkite ["Microsoft Easy Fix" sprendimus](https://support.microsoft.com/topic/microsoft-easy-fix-solutions-have-been-discontinued-b0f4b5f9-3b5a-bd9e-d75d-d45e2f12e16c).

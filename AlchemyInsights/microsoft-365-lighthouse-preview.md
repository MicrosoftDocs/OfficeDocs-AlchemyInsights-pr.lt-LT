# <a name="welcome-to-the-microsoft-365-lighthouse-private-preview"></a>Sveiki! čia "Microsoft 365" Švyturys privati peržiūra

Jūs dalyvaujate privačiame "Microsoft 365" švyturio peržiūroje. Jei norite pateikti atsiliepimų apie "Microsoft 365" švyturį, skelbkite išsamią informaciją mūsų [vartotojo balso svetainėje](https://aka.ms/M365Lighthouseuservoice). Jei turite klausimų arba kyla problemų, kai naudojate "Microsoft 365" švyturį, sukurkite palaikymo užklausą dėl tolesnės pagalbos.

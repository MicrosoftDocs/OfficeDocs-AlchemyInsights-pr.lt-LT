# <a name="upgrade-to-the-full-version-of-microsoft-teams"></a>Versijos naujinimas į visą "Microsoft Teams" versiją

Reikia atnaujinti versiją iš nemokamos į visą "Microsoft Teams" versiją? Peržiūrėkite ["Microsoft Teams" versijos naujinimą nemokamai į "Microsoft 365" prenumeratos versiją](https://docs.microsoft.com/microsoftteams/upgrade-freemium)

Turėsite atitikti šiuos reikalavimus:

- Jūs esate asmuo, prisiregistravęs dėl dabartinės nemokamos "Teams" versijos.
- Jei turite savo domeną, jis dar negali būti susietas su Azure Active Directory.
- Norėdami perkelti duomenis naujinant versiją, turite užsiprenumeruoti atnaujinimo proceso metu. Jei prenumeratą įsigijote ne atnaujinimo proceso metu, duomenų perkelti negalite.

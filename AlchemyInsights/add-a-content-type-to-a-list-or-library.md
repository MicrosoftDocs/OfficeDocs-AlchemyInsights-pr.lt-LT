# <a name="add-a-content-type-to-a-list-or-library"></a>Turinio tipo įtraukimas į sąrašą arba biblioteką

1. Eikite į [sąrašo arba bibliotekos parametrus](https://support.microsoft.com/en-us/office/edit-list-settings-in-sharepoint-online-4d35793b-246e-42a3-990c-563a83795b7f).
2. Dalyje **Turinio tipai**pasirinkite Įtraukti iš **esamų svetainės turinio tipų**. Jei **turinio tipai** nerodomi, pasirinkite **Išplėstiniai parametrai**.
3. Pasirinkite iš sąrašo **Pasirinkite svetainės turinio tipus** iš.
4. Sąraše Galimi svetainės turinio tipai pasirinkite norimo turinio tipo pavadinimą ir perkelkite pasirinktą turinio tipą į sąrašą Įtrauktinų turinio tipų, pasirinkite Įtraukti.
5. Pakartokite veiksmus su papildomais turinio tipais.

Daugiau informacijos rasite [Turinio tipo įtraukimas į sąrašą arba biblioteką](https://support.microsoft.com/en-us/office/add-a-content-type-to-a-list-or-library-917366ae-f7a2-47ad-87a5-9689a1884e60).

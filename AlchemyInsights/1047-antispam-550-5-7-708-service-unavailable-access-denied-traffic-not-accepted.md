# <a name="550-57708-access-denied-traffic-not-accepted-from-this-ip"></a>550 5.7.708 prieiga uždrausta, eismo nebuvo priimtas iš šio IP

Ši klaida įvyksta, kai siunčiant laišką iš žinoma, mažas reputacijos IP adresus, kurie paprastai yra naudojami naujiems klientams.
  
Jei turite siųsti laišką iš šios IP adresų prieš galite įsigyti licencijas, susisiekite su palaikymo u˛klausa išimtį, kol jūs galite įsigyti licencijas.

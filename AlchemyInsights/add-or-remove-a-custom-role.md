# <a name="add-or-remove-a-custom-role"></a>Pasirinktinio vaidmens įtraukimas arba šalinimas

Norėdami valdyti "Azure" išteklius, skaitykite " [Azure" vaidmenų priskyrimų įtraukimas arba šalinimas naudojant "Azure" portalą](https://docs.microsoft.com/azure/role-based-access-control/role-assignments-portal).

Norėdami valdyti "Azure Active Directory" tinkintus vaidmenis ("Azure AD"), peržiūrėkite [pasirinktinio administratoriaus vaidmenis "Azure Active Directory" (peržiūra)](https://docs.microsoft.com/azure/active-directory/users-groups-roles/roles-custom-overview).

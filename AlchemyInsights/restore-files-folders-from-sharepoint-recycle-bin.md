# <a name="restore-files-or-folders-from-the-sharepoint-recycle-bin"></a>Atkurti failus arba aplankus iš SharePoint šiukšlinės 

Panaikinus elementą iš SharePoint dokumentų bibliotekoje arba sąraše, jis nėra iš karto pašalinama iš SharePoint. Panaikinti elementai pereis į svetainės šiukšlinę tam tikrą laiką. Per tą laiką galite atkurti panaikintus elementus pradinėje vietoje.

- [Atkurti failus arba aplankus iš SharePoint šiukšlinės](https://support.office.com/article/Restore-items-in-the-Recycle-Bin-of-a-SharePoint-site-6df466b6-55f2-4898-8d6e-c0dff851a0be)
- [Panaikintų failų arba aplankų atkūrimas "OneDrive"](https://support.office.com/article/restore-deleted-files-or-folders-in-onedrive-949ada80-0026-4db3-a953-c99083e6a84f)

# <a name="give-users-access-to-sharepoint-site"></a>Suteikti vartotojams prieigą prie SharePoint svetainės

Prieigos užklausos funkcija leidžia žmonėms prašyti prieigos prie turinio, kurio jie šiuo metu neturi teisės matyti. Kaip svetainės savininkas, galite konfigūruoti funkciją siųsti jums laiškus, kai kas nors prašo prieigos prie svetainės. Norėdami gauti daugiau informacijos, apsilankykite [Nustatyti ir tvarkyti prieigos užklausas](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3).

Štai keletas bendrų nuorodų, susijusių su prieigos užklausomis:

- [Prieiga uždrausta" prie prieigos užklausų sąrašo arba "Užklausos patvirtinimas nepavyko", kai apdorojant laukia užklausą SharePoint Online](https://docs.microsoft.com/sharepoint/support/sharing-and-permissions/request-approval-failed)

- [Prieigos užklausų parametrų keitimas](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#bk_enableallow)

- [Svetainės prieigos užklausų patvirtinimas arba atmetimas](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#__toc374462558)

- [Kvietimų išoriniams vartotojams tvarkymas (tik "SharePoint Online")](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#__toc334189260)

- [Kvietimo siuntimas iš naujo](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#__toc374462560)

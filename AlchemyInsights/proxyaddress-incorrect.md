# <a name="proxyaddress-incorrect"></a>Neteisingas ProxyAddress

Kai objektas sinchronizuojamas su "Azure AD", "Active Directory" atribute proxyAddresses nurodytos reikšmės lyginamos su "Azure AD" taisyklėmis, o tada atributas proxyAddresses užpildomas "Azure AD". Todėl objekto Active Directory atributo proxyAddresses reikšmės negali būti tokios pačios kaip "Azure AD" atributo proxyAddresses reikšmės.
  
Jei norite sužinoti daugiau apie tai, kaip įvedamas proxyAddress, pažiūrėkite, [kaip atributas proxyAddress yra užpildomas "AZURE ad](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad)".

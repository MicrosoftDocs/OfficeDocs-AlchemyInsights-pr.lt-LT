# <a name="email-forwarding-report"></a>Siųsti ataskaitą

Norėdami peržiūrėti ataskaitą kiek automatiškai peradresuojamų pranešimų šiuo metu organizacijoje, patikrinti [pašto srautą įžvalgos](https://support.office.com/article/beb6acaa-6016-4d54-ba7e-3d6d035e2b46) į ["Office 365" sauga &amp; atitikties užtikrinimo centre](https://protection.office.com/#/homepage).
  
Atkreipkite dėmesį: jei tai pirmas kartas, galite prisijungti prie saugumo &amp; atitikties užtikrinimo centre, jums gali tekti atnaujinti naršyklės po keletą minučių, kad pamatyti visą reikiamą informaciją.

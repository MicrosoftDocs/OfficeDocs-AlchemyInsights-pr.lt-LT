# <a name="how-to-create-a-file-request"></a>Kaip sukurti failo užklausą

"OneDrive" failo užklausos funkcija leidžia pasirinkti aplanką, kuriame kiti gali nusiųsti failus naudodami saitą, kurį siunčiate. Vartotojai, kuriuos prašote failų, negali matyti, kas yra aplanke – jie gali įkelti tik failus.

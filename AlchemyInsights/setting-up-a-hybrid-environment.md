# <a name="setting-up-a-hybrid-environment"></a>Hibridinės aplinkos nustatymas

Kad vartotojai sklandžiai veiktų kartu, nesvarbu, ar jų duomenys saugomi vietiniuose serveriuose arba "Microsoft 365", gali reikėti nustatyti hibridinę aplinką. Norėdami peržiūrėti savo parinktis, peržiūrėkite ["Microsoft 365" integravimą su vietinėmis aplinkomis](https://docs.microsoft.com/office365/enterprise/office-365-integration).

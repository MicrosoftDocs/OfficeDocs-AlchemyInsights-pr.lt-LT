# <a name="setting-up-a-hybrid-environment"></a>Greitojo aplinkoje nustatymas

Jei norite leisti veikti sklandžiai kartu Nesvarbu jei jų duomenys yra saugomi vietiniame serveriuose arba "Office 365" vartotojams, reikia nustatyti greitojo aplinkoje. Norint peržiūrėti jūsų parinktis, peržiūrėkite ["Office 365" integracija su vietinėje aplinkoje](https://support.office.com/article/263faf8d-aa21-428b-aed3-2021837a4b65.aspx).

# <a name="set-passwords-to-never-expire"></a>Nustatyti slaptažodį niekada baigiasi

Nustatyti visus slaptažodžius, niekada baigiasi, "Microsoft" 365 administravimo centro, eikite į **Parametrai** > [saugumo &amp; privatumo](https://portal.office.com/adminportal/home#/settings/security) puslapis. Pasirinkite **Redaguoti** šalia **slaptažodžių strategija**ir įjunkite slankiklį **ant**.
  
Daugiau informacijos rasite: [nustatyti slaptažodžio galiojimo pabaigos strategija organizacijoje.](https://support.office.com/article/0f54736f-eb22-414c-8273-498a0918678f)

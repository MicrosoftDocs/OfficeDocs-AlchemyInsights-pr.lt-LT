# <a name="use-sara-to-diagnose-and-resolve-outlook-authentication-issues"></a>Naudoti SaRA diagnozuoti ir išspręsti Outlook autentiškumo klausimus

Mes rekomenduojame naudoti į ["Outlook" saugo klausia mano slaptažodžio](https://aka.ms/SaRA-OutlookPwdPrompt) diagnostikos pažeistą mašina spręsti problemas tais atvejais, kai "Outlook" nuolat raginama įvesti slaptažodį. Šis [SaRA](https://diagnostics.office.com/#/) diagnostikos ar automatizuoti sutikrinimai ir grąžina galimų sprendimų galite naudoti bet kokį aptiktą problemoms spręsti.

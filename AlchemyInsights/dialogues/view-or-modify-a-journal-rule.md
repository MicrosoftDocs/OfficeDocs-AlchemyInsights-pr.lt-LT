# <a name="view-or-modify-a-journal-rule"></a>Žurnalo taisyklės peržiūra arba modifikavimas

Toliau nurodyta, kaip tai padaryti.

1. " [Exchange" administravimo centre](https://go.microsoft.com/fwlink/p/?linkid=2059104)eikite į **atitikties valdymo**  >  **žurnalų taisykles**.
2. Sąrašo rodinyje matysite visas organizacijos žurnalo taisykles.
3. Dukart spustelėkite taisyklę, kurią norite peržiūrėti arba modifikuoti.
4. Dalyje **žurnalo taisyklė** modifikuokite norimus parametrus.

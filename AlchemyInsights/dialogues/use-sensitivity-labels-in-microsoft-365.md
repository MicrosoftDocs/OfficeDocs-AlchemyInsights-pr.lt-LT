# <a name="use-sensitivity-labels-in-microsoft-365"></a>"Microsoft 365" jautrumo žymių naudojimas

Galite naudoti jautrumo žymes:
- Įgalinti apsaugos parametrus, pvz., šifravimą arba vandenženklius ant etikete pažymėto turinio.
- Apsaugokite turinį "Office" programose skirtingose platformose ir įrenginiuose.
- Apsaugokite turinį trečiųjų šalių programose ir paslaugose naudodami "Microsoft" debesies programų saugą.
- Išplėskite trečiųjų šalių taikomųjų programų ir tarnybų slaptumo žymas.
- Klasifikuokite turinį nenaudodami jokių apsaugos parametrų.

Daugiau informacijos rasite [Sužinokite apie jautrumo žymes](https://docs.microsoft.com/microsoft-365/compliance/sensitivity-labels).

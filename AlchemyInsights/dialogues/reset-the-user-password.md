# <a name="reset-the-users-password"></a>Vartotojo slaptažodžio nustatymas iš naujo

Prieš atlikdami kitus veiksmus turėsite iš naujo nustatyti vartotojo slaptažodį. Toliau nurodyta, kaip tai padaryti.

1. Pasirinkite **vartotojai**  >  **[aktyvūs vartotojai](https://go.microsoft.com/fwlink/p/?linkid=834822)**.
1. Pasirinkite vartotoją, kurio paskyra buvo pažeista.
1. Pasirodžiusiame Iškeliamasis spustelėkite **iš naujo nustatyti slaptažodį**.

> [!NOTE]
> Nesiųskite slaptažodžio vartotojui elektroniniu paštu.

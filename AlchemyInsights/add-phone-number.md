# <a name="add-phone-number"></a>Telefono numerio įtraukimas

Prieš nustatydami savo organizacijos vartotojus, kad jie galėtų skambinti ir priimti skambučius, turite [gauti jiems telefono numerius](https://docs.microsoft.com/MicrosoftTeams/phone-number-calling-plans/port-order-overview). Pateikėme nuoseklų procesą, kad užbaigtumėte telefono sistemos sąranką. Peržiūrėkite [Telefono sistemos nustatymas organizacijoje](https://docs.microsoft.com/MicrosoftTeams/phone-number-calling-plans/port-order-overview).

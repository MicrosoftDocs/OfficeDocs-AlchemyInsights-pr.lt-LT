# <a name="setting-up-visio"></a>"Visio" nustatymas

Atrodo, kad reikia pagalbos nustatant "Visio" naudojant "Microsoft 365" prenumeratą.
  
Pranešimas "Mac" naudotojams: šiuo metu "Visio" produktas nepasiekiamas "MacOS". Jei turite kompiuterį, turėtumėte galėti įdiegti programinę įrangą atlikdami toliau nurodytus veiksmus.
  
Galite atlikti šiame [straipsnyje](https://support.office.com/article/f98f21e3-aa02-4827-9167-ddab5b025710.aspx) nurodytus veiksmus, kad sąranka būtų baigta.

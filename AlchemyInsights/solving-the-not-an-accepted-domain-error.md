# <a name="got-a-parameter-cannot-be-found-error"></a>Nepavyksta rasti parametro?

Jei gaunate klaidos pranešimą **"parametro negalima rasti, kuris atitinka parametro pavadinimą" EmailAddresses "**, tai reiškia, kad" Microsoft 365 "bus šiek tiek ilgesnis, kad būtų galima nustatyti nuomotoją arba pasirinktinio domeno, jei neseniai įtraukėte vieną.
  
Sąrankos procesas gali užtrukti iki 4 valandų. Palaukite, kol procesas nustatymas turi būti baigtas, tada bandykite dar kartą.

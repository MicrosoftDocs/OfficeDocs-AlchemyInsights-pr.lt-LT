# <a name="public-folder-removed-from-outlook-on-the-web-owa-favorites"></a>Viešasis aplankas, pašalintas Outlook žiniatinklio (OWA) parankinių

Yra žinoma problema dėl viešųjų aplankų, kurie pašalinami iš parankinių OWA kiekvieną kartą atsiėjus arba atnaujinus ekraną. Netrukus problema bus išspręsta. Iki to laiko iš naujo įtraukite aplanką, kad būtų galima pasiekti iš OWA.

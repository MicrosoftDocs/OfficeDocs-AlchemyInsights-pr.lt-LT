# <a name="sharing-options-appear-dim-or-are-not-available"></a>Bendro naudojimo parinktys yra neryškias arba nepasiekiamos

Jei kai kurios bendrinimo parinktys arba nepasiekiamos bandant bendrinti iš "SharePoint Online" arba "OneDrive", peržiūrėkite bendrinimo [parinktis yra pilka, kai bendrinate naudodami "SharePoint Online" arba "OneDrive"](https://docs.microsoft.com/sharepoint/support/administration/sharing-options-grayed-out-when-sharing-from-sharepoint-online-or-onedrive).

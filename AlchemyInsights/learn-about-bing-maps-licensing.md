# <a name="learn-about-bing-maps-licensing"></a>Sužinokite apie "Bing" žemėlapių licencijavimą

"Bing" žemėlapių tarnyba siūlo kelias lanksčias licencijavimo parinktis, kurios palaikys jūsų organizacijos poreikius ir techninius reikalavimus. Be to, su nemokama pakopa galite pradėti kurti "Bing Maps" šiandien.

Išsamesnės informacijos ieškokite ["Bing" žemėlapių licencijavimas](https://go.microsoft.com/fwlink/?linkid=2150203).

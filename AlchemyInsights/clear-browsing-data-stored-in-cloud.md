# <a name="clear-the-browsing-data-stored-in-the-cloud-if-youve-turned-on-sync-or-personalization"></a>Išvalyti debesyje saugomus naršymo duomenis (jei esate įjungę sinchronizavimą arba suasmeninimą)

Norėdami peržiūrėti, valdyti arba panaikinti debesyje įrašytus duomenis, naudokite privatumo ataskaitų sritį. Duomenys, kuriuos panaikinsite naudodami ataskaitų sritį, nebus panaikinti iš jūsų įrenginio. Daugiau informacijos [žr. Duomenų rodymas privatumo ataskaitų srityje](https://support.microsoft.com/windows/view-your-data-on-the-privacy-dashboard-03d3e27f-1981-5ff4-ba1c-d6b1031ae433).

# <a name="diagnostic-to-enable-basic-authentication-for-exchange-online-protocols"></a>Diagnostikos, kad įgalintumėte bazinį autentifikavimą Exchange Online protokolams

Naudodami šią diagnostiką, galite įgalinti bazinį "Exchange Online" protokolų, pvz., POP3, IMAP4, Exchange ActiveSync, "Exchange Web Services", autonominės adresų knygelės, MAPI, RPC ir nuotolinės "PowerShell", autentifikavimą, kurį "Microsoft" neseniai išjungė jūsų organizacijoje. 

Siunčiame tiesioginius pranešimus per pranešimų centrą, kad nuomotojai žinotų, kur jie pasirengę išjungti bazinį autentifikavimą savo aplinkoje dėl nenaudingo naudojimo, o tai padės apsaugoti aplinką nuo susijusių saugos pavojų.

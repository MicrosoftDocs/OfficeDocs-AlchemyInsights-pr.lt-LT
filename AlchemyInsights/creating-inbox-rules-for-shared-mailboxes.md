# <a name="creating-inbox-rules-for-shared-mailboxes"></a>Bendrai naudojamos pašto dėžutės aplanko Gauta taisyklės kūrimas

Galite pridėti taisykles bendrinamą pašto dėžutę turite prieigą prie panašiai kaip įtraukti taisykles į savo sąskaita. 
  
Įsitikinkite, kad esate narys atitinkamų bendrai naudojamą pašto dėžutę, ir tada atlikite veiksmus šiame straipsnyje: [taisykles įtraukti bendrinamą pašto dėžutę](https://support.office.com/article/b0963400-2a51-4c64-afc7-b816d737d164 )

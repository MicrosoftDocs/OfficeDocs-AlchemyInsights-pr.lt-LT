# <a name="site-is-not-available"></a>Svetainė neprieinama

Jei SharePoint arba "OneDrive" svetainė neprieinama keliems vartotojams, gali būti laikinai tarnybos problema. Patikrinkite [tarnybos sveikatos ataskaitų srities](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) Norėdami pamatyti, jei jūsų nuomininkas turi įtakos.

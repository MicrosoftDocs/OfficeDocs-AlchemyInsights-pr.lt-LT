# <a name="site-is-not-available"></a>Svetainės nėra

Jei jūsų SharePoint arba "OneDrive" svetainėje nėra keliems vartotojams, gali būti laikinai skundams. Patikrinkite[Tarnybų sveikatos stebėjimo skydas](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) nuorodą matysite, jeigu jūsų nuomininkas smūgiuoja.

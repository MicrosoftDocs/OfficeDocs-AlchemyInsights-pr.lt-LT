# <a name="how-to-install-mac-office-from-the-portal"></a>Kaip įdiegti "Mac Office" iš portalo

1. [Pašalinkite "Office 2011 for Mac" iš savo kompiuterių](https://support.office.com/article/4bfcd230-0ea1-4656-bf30-dbfa44d358fa?wt.mc_id=Alchemy_ClientDIA).

2. [Programinės įrangos atsisiuntimo parametrų valdymas](https://docs.microsoft.com/DeployOffice/manage-software-download-settings-office-365).

3. [Priskirti licencijas vartotojams](https://docs.microsoft.com/microsoft-365/admin/subscriptions-and-billing/assign-licenses-to-users).

4. Tada vartotojai gali patys įdiegti "Office" programinę įrangą arba galite naudoti " [Office" diegimo "Mac" vartotojams](https://docs.microsoft.com/DeployOffice/mac/deployment-guide-for-office-for-mac).

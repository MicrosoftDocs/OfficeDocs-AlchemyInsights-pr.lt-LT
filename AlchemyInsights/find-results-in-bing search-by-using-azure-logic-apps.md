# <a name="find-results-in-bing-search-by-using-azure-logic-apps"></a>"Bing" ieškos rezultatų radimas naudojant "Azure" logikos taikomąsias programas

Naudodami "Bing" iešką iš logikos taikomosios programos, galite automatizuoti ieškos rezultatų apdorojimą ir atlikti kitus veiksmus. Pavyzdžiui, galite ieškoti naujienų elementų, tada "Twitter" publikuokite juos kaip "Twitter" savo informacijos santraukoje.

Jei norite sužinoti daugiau, peržiūrėkite [rasti rezultatus "Bing" paieškoje naudojant "Azure" logikos taikomąsias programas](https://go.microsoft.com/fwlink/?linkid=2151928).

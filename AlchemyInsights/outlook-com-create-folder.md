# <a name="create-a-folder"></a>Sukurkite aplanką

1. Aplanko srities apačioje pažymėkite **naują aplanką**. Jums gali tekti slinkti žemyn, kad pamatytumėte mygtuką **Naujas aplankas** aplankų srityje.
2. Teksto lauke įveskite aplanko pavadinimą ir paspauskite klavišą Enter.

Norėdami sužinoti daugiau, peržiūrėkite [Kurti aplanką programoje "Outlook.com"](https://support.office.com/article/6bb0723a-f39f-4a8d-bb3f-fab5dcc2510a?wt.mc_id=Office_Outlook_com_Alchemy).

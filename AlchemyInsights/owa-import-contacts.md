# <a name="import-contacts"></a>Importuoti kontaktus

Prieš importuodami kontaktus į "Outlook" internete, turėsite juos eksportuoti iš kitų el. pašto teikėjo kaip .csv failą. Tada atlikite žemiau pateiktus nurodymus.

1. Puslapio apačioje pasirinkite piktogramą žmonės eiti į puslapį žmonės.
2. Įrankių juostoje pasirinkite **valdyti** > **Importuoti kontaktus**.
3. Pasirinkite **Naršyti**, pasirinkite CSV failą ir pasirinkite **atidaryti**.
4. Pasirinkite **importuoti**.

Norėdami sužinoti daugiau, peržiūrėkite [adresatų importavimas į programą Outlook](https://support.office.com/article/bb796340-b58a-46c1-90c7-b549b8f3c5f8#ID0EAACAAA=Outlook_on_the_web).

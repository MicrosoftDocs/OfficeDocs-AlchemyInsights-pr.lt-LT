# <a name="upload-process-overview-in-microsoft-stream"></a>Įkelti proceso apžvalgą programoje "Microsoft Stream"

Kai įkeliate vaizdo įrašą, "Microsoft Stream" automatiškai užkoduoja jį į kelis bitrates, kad vaizdo įrašą būtų galima transliuoti naudojant adaptyviąją srautinio bitų srautinio perdavimo. "Microsoft Stream" optimizuoja vaizdo įrašo bitrate ir kokybę, atsižvelgdama į žiūrimačio pralaidumą.

- [Įkelti proceso apžvalgą programoje "Microsoft Stream"](https://docs.microsoft.com/stream/upload-process-overview)

# <a name="remove-a-hub-site"></a>Koncentratoriaus svetainės pašalinimas

Norėdami išregistruoti telkinio svetainę naujame SharePoint administravimo centre:
  
1. Naujojo SharePoint administravimo centro dalyje **svetainės**spustelėkite **aktyvios svetainės**.

2. Pasirinkite svetainę, spustelėkite **telkinio svetainė**, tada spustelėkite **išregistruoti kaip telkinio svetainę**.

3. Spustelėkite **Gerai**.

[Sužinokite daugiau apie koncentratoriaus svetaines](https://support.office.com/article/what-is-a-sharepoint-hub-site-fe26ae84-14b7-45b6-a6d1-948b3966427f)

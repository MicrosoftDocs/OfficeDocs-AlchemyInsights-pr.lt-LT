# <a name="remove-a-hub-site"></a>Pašalinti koncentratoriaus svetainė

Išregistruoti naują SharePoint administravimo centro centrinėje svetainėje
  
1. Nauja SharePoint administravimo centro, pagal **svetaines**, spustelėkite **aktyvus svetainės**. 
    
2. Pasirinkite svetainę, spustelėkite **koncentratoriaus svetainė**, tada **išregistravimo kaip koncentratoriaus svetainė**. 
    
3. Spustelėkite **gerai**. 
    
[Sužinokite daugiau apie habų svetainėse](https://support.office.com/article/what-is-a-sharepoint-hub-site-fe26ae84-14b7-45b6-a6d1-948b3966427f?ui=en-US&amp;rs=en-US&amp;ad=US)

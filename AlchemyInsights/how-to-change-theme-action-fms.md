# <a name="how-to-change-your-theme"></a>Kaip pakeisti savo temą

Norėdami keisti temą "Outlook.com" arba "Outlook" internete, eikite į **Parametrai** ir pasirinkite temą pagal **temą**. Norėdami peržiūrėti visas galimas temas, pasirinkite **Peržiūrėti viską** pagal **temą**.

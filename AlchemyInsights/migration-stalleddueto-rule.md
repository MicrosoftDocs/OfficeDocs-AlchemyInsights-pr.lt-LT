# <a name="stalleddueto-errors-during-migration"></a>"StalledDueTo" klaidų perkėlimo metu

Galite matyti įvairių klaidų, kurios prasideda "StalledDueTo..." perkeldami į "Exchange Online". Daugelis šių klaidų yra visiškai įprastos. Daugiau informacijos ieškokite [pagal išteklių buferizavimo ir prioritetų nustatymą "Exchange Online" perkėlimų](https://techcommunity.microsoft.com/t5/exchange-team-blog/resource-based-throttling-and-prioritization-in-exchange-online/ba-p/608020).

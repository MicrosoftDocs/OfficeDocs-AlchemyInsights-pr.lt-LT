# <a name="how-to-type-special-characters-on-a-mac"></a>Kaip įvesti specialiuosius simbolius "Mac"

Norėdami įvesti specialiuosius simbolius savo "Mac", atidaryti **specialiųjų simbolių** meniu su klaviatūros klavišai ⌘ **komandą** + **valdymo** + **vietos**.

Sparčioji nuoroda dėl @ taip pat **ALT** + **64**.

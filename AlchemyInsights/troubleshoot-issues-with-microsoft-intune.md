# <a name="troubleshoot-issues-with-microsoft-intune"></a>Šalinkite triktis naudodami Microsoft Intune

Peržiūrėti išteklių spręsti savo problemą dabar žemiau.
  
1. Naudoti [Intune trikčių šalinimo portalas](https://devicemanagement.microsoft.com/#blade/Microsoft_Intune_DeviceSettings/TroubleshootBlade) diagnozuoti ir išspręsti kai dažni gedimai. Peržiūrėti [šį](https://docs.microsoft.com/intune/help-desk-operators) dokumentą daugiau detalių.  
2. Peržiūrėti [Intune TechNet ](https://social.technet.microsoft.com/forums/home?forum=microsoftintuneprod)rasti atsakymus ir dažnų problemų sprendimus.  
3. Peržiūrėkite [Paslaugų sveikatos](https://portal.office.com/AdminPortal/Home#/servicehealth) pamatyti paslaugos būseną.   
4. Peržiūrėti [Intune dokumentų](https://docs.microsoft.com/intune/) planavimo, how-to, trikčių diagnostikos straipsniuose ir daugiau.

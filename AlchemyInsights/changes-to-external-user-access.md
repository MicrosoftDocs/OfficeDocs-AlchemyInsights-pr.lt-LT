# <a name="changes-to-external-user-access"></a>Išorinių vartotojų prieigos pokyčiai

Nuo kovo 23 d., 2018, Išoriniai vartotojai nebegalės pasiekti turinio, kuris bendrinamas su visais, visiems autentifikuotiems vartotojams arba visiems formų vartotojams, net jei jie buvo pakviesti arba pridėti įgaliotoms grupėms. Išoriniai vartotojai galės pasiekti turinį, kuris buvo bendrinamas su vartotoju arba su grupėmis, kurioms priklauso vartotojas. Jei norite išsaugoti esamus parametrus, žiūrėkite: https://support.microsoft.com/help/4089534/how-to-grant-the-everyone-claim-to-external-users-in-office-365

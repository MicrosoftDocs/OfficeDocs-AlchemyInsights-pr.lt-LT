# <a name="delegated-mailbox-permissions-in-a-hybrid-environment"></a>Įgaliotosios pašto dėžutės teisės greitojoje aplinkoje

Atsižvelgiant į "Exchange" versiją, kurią įdiegėte vietinėje organizacijoje, gali reikėti atlikti keletą papildomų veiksmų, kad nustatytumėte įgaliotosios pašto dėžutės teises tarp vietinės organizacijos ir "Exchange Online" "Microsoft 365". Jei norite gauti daugiau informacijos apie tai, kaip nustatyti įgaliotąsias teises, peržiūrėkite [Konfigūruoti Exchange palaikyti įgaliotasis pašto dėžutės teises hibridinis diegimas](https://technet.microsoft.com/library/mt784505%28v=exchg.150%29.aspx).

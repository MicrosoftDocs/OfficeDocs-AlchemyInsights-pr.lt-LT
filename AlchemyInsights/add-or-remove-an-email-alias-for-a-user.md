# <a name="add-or-remove-an-email-address-for-a-user"></a>Vartotojo el. pašto adreso pridėjimas arba pašalinimas

[Atlikite šiuos veiksmus](https://portal.office.com/AdminPortal/Home#/AssistedGuide/addemailoptions) Microsoft 365 administravimo centro pradėti.

 **Pastaba**: jūs galite turėti daugiau nei vieną elektroninio pašto adresą vartotojui, bet papildomų *elektroninio pašto slapyvardžius* galima naudoti tik gauti el. Jei jums reikia siųsti el. laišką su pseudonimu, sukurkite [bendrinamą pašto dėžutę](https://support.office.com/article/871a246d-3acd-4bba-948e-5de8be0544c9). Tokiu būdu gausite pseudonimą siųsti ir gauti laiškus.

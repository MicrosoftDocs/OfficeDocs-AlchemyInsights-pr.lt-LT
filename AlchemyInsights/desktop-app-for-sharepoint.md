# <a name="desktop-app-for-sharepoint"></a>Kompiuterio taikomoji programa, skirta "SharePoint"

Jūsų organizacija įvedama į "SharePoint" kompiuterio taikomosios programos privačią peržiūros programą. "Microsoft" šiuo metu nepalaiko "SharePoint" kompiuterio taikomosios programos (privačios peržiūros). Naudokite savo išvardytus "Microsoft Private Preview" partnerius, kad gautumėte pagalbos naudojant kompiuterio taikomąją programą, skirtą "SharePoint".

# <a name="desktop-app-for-sharepoint"></a>Darbalaukio programėlė, skirta "SharePoint"

Jūsų organizacija įvedama į "SharePoint" darbalaukio programėlės privataus peržiūros programą. "Microsoft" šiuo metu nepalaiko darbalaukio programėlę, skirtą "SharePoint" (privati peržiūra). Norėdami gauti pagalbos naudodami darbalaukio programėlę, skirtą "SharePoint", naudokite savo "Microsoft" privatųjį peržiūros partnerius.

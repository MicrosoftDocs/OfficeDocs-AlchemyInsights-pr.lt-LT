# <a name="outlook-error-cannot-start-microsoft-outlook"></a>"Outlook" klaida: nepavyksta paleisti "Microsoft Outlook"

Kyla problemų pradedant "Outlook 2016", "Outlook 2013" arba "Outlook 2010" arba "Microsoft Outlook". Nepavyksta atidaryti "Outlook" lango "klaidų pranešimo?
  
Pirmiausia [šiame straipsnyje](https://support.office.com/article/I-can-t-start-Microsoft-Outlook-2016-2013-or-2010-or-receive-the-error-Cannot-start-Microsoft-Office-Outlook-Cannot-open-the-Outlook-Window-d1f69da6-b333-4650-97bf-4d77bd7abb85) pateiktus trikčių diagnostikos veiksmus galite naudoti Norėdami išspręsti šias paleisties problemas. 
  
Taip pat galite paleisti [Išplėstinio diagnostikos įrankį](https://aka.ms/SaRA-OutlookAdvDiagnostics) paveiktame kompiuteryje.

# <a name="cant-add-or-change-teams-profile-picture"></a>Negalima įtraukti arba keisti komandų profilio paveikslėlio

Norėdami įtraukti arba keisti profilio paveikslėlį, vartotojai turi būti visiškai įgalinti:

- "Exchange Online"
- "SharePoint Online"
- Galimybė kurti "Microsoft" 365 grupes

Peržiūrėkite savo komandos administratorių apie tai, kaip gauti reikiamas teises. Instrukcijų ieškokite [paveikslėlio keitimas naudojant "teams"](https://support.microsoft.com/office/change-your-picture-in-teams-7a711943-9248-420e-b814-c071aa8d9b9c).

# <a name="set-microsoft-edge-as-the-default-browser-on-a-windows-10-device"></a>"Microsoft Edge" nustatymas kaip numatytosios naršyklės "Windows 10" įrenginyje

Toliau nurodyta, kaip tai padaryti.

1. Pasirinkite mygtuką **Pradžia,** tada įveskite **Numatytosios programos**.

1. Ieškos rezultatuose pasirinkite **Numatytosios programos**.

1. Dalyje **Žiniatinklio naršyklė** pasirinkite **"Microsoft Edge".**

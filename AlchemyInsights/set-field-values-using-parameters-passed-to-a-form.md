# <a name="set-field-values-using-parameters-passed-to-a-form---lookup-fields"></a>Laukų reikšmių nustatymas naudojant į formą perduotus parametrus – peržvalgos laukai

Nustatykite peržvalgos laukų reikšmę ir pavyzdys: naudokite Xrm.Utility.openEntityForm, kad atidarytumėte naują langą. Negalite nustatyti šalių sąrašo arba peržvalgų reikšmių.

Daugiau informacijos [ieškokite Set values for lookup fields](https://docs.microsoft.com/previous-versions/dynamicscrm-2016/developers-guide/gg334375(v=crm.8)#set-values-for-lookup-fields).

# <a name="enable-seamless-sso"></a>Sudaryti vientisas SSO

Žydros Active Directory (Azure AD) besiūlių bendrosios autentifikacijos (SSO besiūlių) automatiškai prisijungia vartotojams kai jie dėl savo verslo darbalaukius, kad esate prisijungę prie įmonės tinklo.
  
[Atlikite veiksmus šiame straipsnyje, kad besiūlių SSO](https://docs.microsoft.com/azure/active-directory/connect/active-directory-aadconnect-sso-quick-start)

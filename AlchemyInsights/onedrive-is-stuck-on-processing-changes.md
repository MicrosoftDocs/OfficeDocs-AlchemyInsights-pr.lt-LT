# <a name="onedrive-is-stuck-on-processing-changes"></a>"OneDrive" užstrigo apdorojant pakeitimus

Jei "OneDrive" ilgą laiką rodo "Pakeitimų apdorojimas", gali būti, kad esate atidarytas internetiniame faile, eilėje įstrigo daug failų arba labai didelis failų sinchronizavimas. Skaityti ["OneDrive" užstrigo "Apdoroti pakeitimus"](https://support.office.com/article/onedrive-is-stuck-on-processing-changes-b386b813-9b66-4e47-8c4c-2b45533edccd) siūlomų veiksmų ir daugiau informacijos.

Jei reikia daugiau pagalbos, galite [el. paštu išsiųsti "OneDrive" palaikymo tarnybą](https://go.microsoft.com/fwlink/p/?LinkId=528676).

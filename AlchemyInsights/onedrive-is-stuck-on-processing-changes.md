# <a name="onedrive-is-stuck-on-processing-changes"></a>"OneDrive" užstrigo apdorojant keitimą

Jei "OneDrive" rodo "apdorojimo pakitimus" ilgą laiką, taip gali būti dėl to, kad esate atidarę internetinį failą, daug failų įstrigo eilėje arba labai daug sinchronizuojamas failas. Skaityti "  [OneDrive" yra užstrigę](https://support.office.com/article/onedrive-is-stuck-on-processing-changes-b386b813-9b66-4e47-8c4c-2b45533edccd) siūlomų veiksmų ir daugiau informacijos "apdorojimo pakeitimai".

Jei reikia daugiau pagalbos, galite  [Siųsti "OneDrive" palaikymą](https://go.microsoft.com/fwlink/p/?LinkId=528676).

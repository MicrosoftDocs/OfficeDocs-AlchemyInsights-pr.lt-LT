# <a name="a-user-has-two-mailboxes"></a>Kai vartotojas turi dvi dėžutes

Hibridinis aplinkoje, kurios naudoja Azure Active Directory prisijungti (AAD jungtis) arba DirSync netyčia gali sukelti vartotojui turi dvi dėžutes: vieną į vietinę organizaciją, o kitą – nuotolinių išteklių saugykloje.
  
Norėdami išspręsti šią problemą, pamatyti [Nuolat išvalyti ankstesnį pašto dėžutės Info](https://blogs.technet.microsoft.com/exchange/2018/01/17/permanently-clear-previous-mailbox-info/).

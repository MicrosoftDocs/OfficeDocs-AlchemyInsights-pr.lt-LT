# <a name="profile-property-synchronization"></a>Profilio ypatybių sinchronizavimas

Negalima susieti skelbimų importavimo papildomų ypatybių, kad būtų galima sinchronizuoti "Azure Active Directory" į vartotojo profilio taikomąją programą. SKELBIMŲ importavimas apsiriboja iš anksto sukonfigūruotais ypatybių rinkiniu, kad būtų garantuotas nuoseklus laikmačio užduoties našumas.
  
[Sužinokite daugiau apie sinchronizuojamas profilio ypatybes](https://go.microsoft.com/fwlink/?linkid=875671)

# <a name="sync-warning-last-synced-more-than-3-days-ago"></a>Sinchronizavimo įspėjimas: paskutinį kartą sinchronizuota daugiau nei prieš 3 dienas

Jei turite papildomų trikčių šalinimo [problemų, žiūrėkite katalogų sinchronizavimo problemų sprendimas](https://docs.microsoft.com/office365/enterprise/fix-problems-with-directory-synchronization).

Galite patikrinti sinchronizavimo būseną "Office" administravimo portale, vykdydami instrukcijas [rodinyje katalogų sinchronizavimo būsena](https://docs.microsoft.com/office365/enterprise/view-directory-synchronization-status).

# <a name="sync-warning-last-synced-more-than-3-days-ago"></a>Sinchronizavimo įspėjimas: paskutinį kartą sinchronizuota daugiau nei prieš 3 dienas

Jei reikia papildomos trikčių diagnostikos, [peržiūrėkite "Office 365" katalogų sinchronizavimo problemų sprendimas](https://docs.microsoft.com/office365/enterprise/fix-problems-with-directory-synchronization).

Sinchronizavimo būseną "Office" administravimo portale galite patikrinti vadovaudamiesi instrukcijomis, pateiktomis ["Office 365" katalogų sinchronizavimo būsenos rodinyje.](https://docs.microsoft.com/office365/enterprise/view-directory-synchronization-status)

# <a name="how-to-select-all-in-outlookcom"></a>Kaip "Pažymėti viską", "Outlook.com"

Jei žymite visas, jei visi pranešimai yra matomas. Taip siekiama išvengti atsitiktinio veiksmus su žinutėmis, jūs negalite matyti.

**CTRL + spustelėkite** atrenka kelis atskirus pranešimus.

**SHIFT + spustelėkite** pažymi visus pranešimus tarp pirmą ir paskutinį pranešimą, pasirinkite.

# <a name="join-a-team-by-code"></a>Prisijungimas prie komandos pagal kodą

"Microsoft" žino apie problemą, kai vartotojai negali kurti prisijungimo prie **komandos** kodo, arba prisijungti prie komandos naudodami funkciją **prisijungti pagal kodą** . Kaip sprendimo būdas, jei reikia, komandos savininkai gali neautomatiškai įtraukti vartotojų.

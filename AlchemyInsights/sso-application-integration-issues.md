# <a name="sso-application-integration-issues"></a>SSO taikomosios programos integravimo problemos

Ieškokite konkrečios taikomosios programos, kurią norite integruoti su vientiso vienkartinio prisijungimo (SSO) rinkiniu, naudodami " [Azure Active Directory" taikomąsias programas](https://docs.microsoft.com/azure/active-directory/saas-apps/tutorial-list) ir vykdykite integravimo trikčių diagnostikos nurodymus.

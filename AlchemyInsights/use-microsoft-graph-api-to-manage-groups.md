# <a name="use-microsoft-graph-api-to-manage-groups"></a>"Microsoft Graph API" naudojimas grupėms valdyti

Grupės yra vartotojų ir kitų vadovų, kurie bendrina prieigą prie "Microsoft" tarnybų arba programos išteklių, rinkiniai. "Microsoft Graph" pateikia API, kuriuos galite naudoti kurdami ir valdydami skirtingus grupių ir grupių funkcijų tipus pagal jūsų scenarijų. Visos su grupe susijusios operacijos programoje "Microsoft Graph" reikalauja administratoriaus sutikimo. Daugiau informacijos ieškokite [darbas su grupėmis programoje "Microsoft Graph"](https://docs.microsoft.com/graph/api/resources/groups-overview).

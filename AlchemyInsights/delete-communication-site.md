# <a name="delete-a-communication-site"></a>Naikinti svetainės, komunikacijos

1. Eikite į svetainę.
    
2. Spustelėkite nustatymų piktogramą viršutiniame dešiniajame kampe.
    
3. Spustelėkite **Svetainės parametrai**. Pagal **Svetainės informaciją**, spustelėkite **panaikinti šią svetainę**, ir tada spustelėkite **Naikinti**.

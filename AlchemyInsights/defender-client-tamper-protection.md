# <a name="defender-tamper-protection"></a>Gynėjas tamper apsaugos 

Naudojant "tamper Protection" kenkėjiškoms programoms neleidžiama imtis tokių veiksmų:

- Virusų ir grėsmių apsaugos išjungimas
- Apsaugos realiuoju laiku išjungimas
- Veikimo būdo stebėjimo išjungimas
- Išjungti antivirusinę (pvz., IOfficeAntivirus (IOAV))
- Debesies teikiamos apsaugos išjungimas
- Šalinami saugos įžvalgų naujinimai

Norėdami gauti daugiau informacijos, žiūrėkite [apsaugoti apsaugos parametrus nuo tamper apsaugos](https://docs.microsoft.com/windows/security/threat-protection/windows-defender-antivirus/prevent-changes-to-security-settings-with-tamper-protection).

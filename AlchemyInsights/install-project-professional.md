# <a name="install-project-professional"></a>Įdiekite "Project Professional"

Žr [įdiegti projekto](https://support.office.com/article/install-project-7059249b-d9fe-4d61-ab96-5c5bf435f281) sužinoti, kaip įdiegti prenumerata versija Project Online, ir ne prenumeratos versija Project 2019, 2016, arba 2013. 

Jei projektas 2010 arba 2007, atlikite veiksmus, nurodytus " [Office 2010](https://support.office.com/article/install-office-2010-1b8f3c9b-bdd2-4a4f-8c88-aa756546529d) " arba " [Office 2007](https://support.office.com/article/install-office-2007-88a8e329-3335-4f82-abb2-ecea3e319657)". 

Projektas parduodamas kaip atskira programa, todėl, jei jo dar neturite, [įsigykite arba Išbandykite "Project](https://products.office.com/project)".

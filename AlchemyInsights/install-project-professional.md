# <a name="install-project-professional"></a>"Project Professional" diegimas

Peržiūrėkite [diegimo projektą](https://support.office.com/article/install-project-7059249b-d9fe-4d61-ab96-5c5bf435f281) ir Sužinokite, kaip įdiegti "Project Online" prenumeratos versiją ir neprenumeruojamą "Project 2019", "2016" arba "2013" versiją. 

Jei turite "Project 2010" arba "2007", atlikite veiksmus, nurodytus " [office 2010](https://support.office.com/article/install-office-2010-1b8f3c9b-bdd2-4a4f-8c88-aa756546529d)"   arba "Office" [2007](https://support.office.com/article/install-office-2007-88a8e329-3335-4f82-abb2-ecea3e319657). 

Projektas parduodamas kaip atskira programa, todėl, jei jos dar neturite, [įsigykite arba Išbandykite projektą](https://products.office.com/project).

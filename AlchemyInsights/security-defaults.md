# <a name="security-defaults"></a>Numatytosios saugos reikšmės

Informacijos apie saugos numatytąsias reikšmes rasite [kas yra saugos numatytosios reikšmės?](https://docs.microsoft.com/azure/active-directory/conditional-access/concept-conditional-access-security-defaults).

"Azure Active Directory" numatytoji saugos reikšmė ("Azure AD") padeda apsaugoti jūsų organizaciją ir ją apsaugoti. Saugos numatytose yra iš anksto sukonfigūruoti bendrųjų atakų saugos parametrai.

"Microsoft" saugos numatytuosius parametrus galima naudoti visiems. Tikslas – užtikrinti, kad visos organizacijos turėtų bazinį saugos lygį be jokių papildomų išlaidų. Galite įjungti saugos numatytąsias reikšmes "Azure" portale.

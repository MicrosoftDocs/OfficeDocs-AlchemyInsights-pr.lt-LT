# <a name="disable-network-scan"></a>Išjungti tinklo nuskaitymą

Tinklo bendrinimo skenavimas gali turėti įtakos našumui.  Norėdami užtikrinti, kad klientas nenuskaito tinklo bendrinimo/failų pagal numatytuosius parametrus, sukonfigūruokite šiuos "Windows" sargybos taikomosios programos parametrus kaip **True**:

- DisableScanningMappedNetworkDrivesForFullScan
- DisableScanningNetworkFiles

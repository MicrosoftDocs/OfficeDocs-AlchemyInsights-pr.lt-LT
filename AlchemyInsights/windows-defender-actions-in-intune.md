# <a name="windows-defender-actions-in-intune"></a>"Windows" sargybos veiksmai intune

"Intune" gali būti naudojamas norint suaktyvinti nuskaitymo pareikalavus ir virusų kodų naujinimą, skirtą "Windows" sargybai atskiruose įrenginiuose.

Sėkmingai suaktyvinus nuotolinį veiksmą, "Windows" sargybos įvykių žurnale atsispindi veikla.

"Windows Endpoint Protection" strategija leidžia "Intune" sukurti papildomus "Windows" sargybos funkcijų parametrus ir taikyti įrenginių rinkiniams.

Daugiau informacijos apie "Windows" sargybos veiksmų suaktyvinimą [ieškokite Configure and run on-demand Microsoft Defender Antivirus scans](https://docs.microsoft.com/windows/security/threat-protection/windows-defender-antivirus/run-scan-windows-defender-antivirus).

# <a name="entity-type-not-showing-in-search-results"></a>Objekto tipas nerodomi ieškos rezultatuose

Jei objekto tipas turi būti rodomas "Dynamics 365" bendrosios sąsajos ieškos kategorijai, ji pirmiausia turi būti įtraukta į programos modulį.

Pastaba: ieškant priedų aktualumo ieška, priedo subjektas turi būti įtrauktas į programos modulį, net jei pirminis įrašas yra tas, kuris rodomas rezultatuose.

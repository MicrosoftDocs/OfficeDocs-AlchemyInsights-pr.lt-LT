# <a name="entity-type-not-showing-in-search-results"></a>Objekto tipas nėra rodomas paieškos rezultatuose

Subjekto tipas rodytų suskirstyti paieškos arba aktualumas search Dynamics 365 vieningas sąsajos, turi būti pirmasis papildomas programos modulis.

Pastaba: Kai ieškote priedų tinkamumą paieškos, arešto subjektas įrašoma į programos modulis nors pirminis įrašas yra, kas turi būti rodoma rezultatus.

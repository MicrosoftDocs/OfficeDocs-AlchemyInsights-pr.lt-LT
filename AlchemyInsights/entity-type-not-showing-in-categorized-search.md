# <a name="entity-type-not-showing-in-search-results"></a>Objekto tipas nerodomas ieškos rezultatuose

Kad objekto tipas būtų rodomas "Dynamics 365" vieningosios sąsajos paieškoje pagal kategorijas arba tinkamumo, jis pirmiausia turi būti įtrauktas į programos modulį.

Pastaba: ieškant atitikimo paieškos priedų, priedo objektas turi būti įtrauktas į programos modulį, net jei pirminis įrašas rodomas rezultatuose.

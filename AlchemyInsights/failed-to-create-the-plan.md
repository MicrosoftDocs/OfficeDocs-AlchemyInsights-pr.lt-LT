# <a name="failed-to-create-the-plan-error-when-trying-to-create-a-planner-tab-in-microsoft-teams"></a>Klaida "nepavyko sukurti plano" bandant kurti "Microsoft teams" planavimo priemonę

Jei gaunate klaidos pranešimą "nepavyko sukurti plano" bandant kurti "Microsoft teams" planavimo priemonę, įsitikinkite, kad administratorius yra grupės narys atlikdami šiuos veiksmus:

1. Administravimo centre eikite į **grupių**  >  [grupės](https://admin.microsoft.com/Adminportal/Home?source=applauncher#/groups). 
2. Pažymėkite grupę, tada skirtuke **nariai** pasirinkite **Peržiūrėti visus ir valdyti narius**.
3. Įtraukite narį ir pasirinkite **įrašyti**.

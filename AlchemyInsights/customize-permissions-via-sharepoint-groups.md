# <a name="how-to-customize-sharepoint-site-permissions-via-sharepoint-groups"></a>Kaip tinkinti SharePoint svetainės teises per SharePoint grupes 

SharePoint grupė – tai vartotojų, kurie turi vienodas teises į svetaines ir turinį, rinkinys. Užuot priskyrę leidimus vienam asmeniui vienu metu, galite naudoti grupes, kad galėtumėte patogiai priskirti tą patį teisių lygį daugeliui žmonių vienu metu.

["SharePoint" svetainės teisių tinkinimas](https://docs.microsoft.com/sharepoint/customize-sharepoint-site-permissions)

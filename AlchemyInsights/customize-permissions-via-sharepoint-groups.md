# <a name="how-to-customize-sharepoint-site-permissions-via-sharepoint-groups"></a>Kaip tinkinti "SharePoint" svetainės teises naudojant "SharePoint" grupes 

"SharePoint" grupė yra vartotojų rinkinys, kurio visi turi tas pačias teises į svetaines ir turinį. Užuot priskirti teises vienam asmeniui vienu metu, galite naudoti grupes, kad galėtumėte patogiai priskirti tą patį teisių lygį daugeliui žmonių vienu metu.

["SharePoint" svetainės teisių tinkinimas](https://docs.microsoft.com/sharepoint/customize-sharepoint-site-permissions)

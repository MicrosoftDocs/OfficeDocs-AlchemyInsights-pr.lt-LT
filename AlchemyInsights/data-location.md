# <a name="data-location"></a>Duomenų vieta

Galite peržiūrėti savo nuomotojo vietą administravimo centre apsilankę [ **organizacijos profilio** puslapyje](https://admin.microsoft.com/AdminPortal/Home#/Settings/OrganizationProfile) ir pasirinkę **duomenų vietą**.

Jei nesate užsiprenumeravę taikomųjų programų, Sužinokite, [kur saugomi jūsų "Microsoft" 365 klientų duomenys](https://docs.microsoft.com/office365/enterprise/o365-data-locations).

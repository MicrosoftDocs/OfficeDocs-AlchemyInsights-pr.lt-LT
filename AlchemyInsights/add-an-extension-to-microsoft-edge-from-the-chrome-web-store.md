# <a name="add-an-extension-to-microsoft-edge-from-the-chrome-web-store"></a>Plėtinio pridėjimas į "Microsoft Edge" iš "Chrome" internetinės parduotuvės

Norėdami įtraukti plėtinį į "Microsoft Edge" iš "Chrome" internetinės parduotuvės, atlikite šiuos veiksmus:

1. Eikite į "Chrome" žiniatinklio parduotuvę.
2. Puslapio viršuje esančioje juostoje pasirinkite **leisti plėtinius iš kitų parduotuvių**.
3. Pasirinkite plėtinį, kurį norite įtraukti, tada spustelėkite **įtraukti į "Chrome"**.
4. Patvirtinimo dialogo lange atidžiai Peržiūrėkite keitimus, kuriuos gali atlikti plėtinys, tada spustelėkite **įtraukti plėtinį**.
Galutinis dialogo langas patvirtins, kad plėtinys buvo pridėtas.

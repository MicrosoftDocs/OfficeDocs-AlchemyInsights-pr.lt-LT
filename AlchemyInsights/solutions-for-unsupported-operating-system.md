# <a name="solutions-for-unsupported-operating-system"></a>Sprendimai "nepalaikomos operacinės sistemos"

Kai bandote įdiegti "Microsoft 365", gali būti rodomas **nepalaikomas operacinės sistemos** klaidos pranešimas, jei įrenginys neatitinka [sistemos reikalavimų](https://products.office.com/office-system-requirements).
  
"Microsoft 365" reikia:
  
- Asmeniniame kompiuteryje: "Windows 10", "Windows 8,1", "Windows 8", "Windows 7" 1 pakeitimų paketas, "Windows Server 2016", "Windows Server 2012 R2", "Windows 2008 2012 Server"

    ARBA

- "Mac OS X 10,10"

Jei gavote šį pranešimą bandydami įdiegti "Office" mobiliajame įrenginyje arba "Chromebook" įrenginyje, Sužinokite, [kaip įdiegti ir paleisti "Microsoft Office" "Chromebook"](https://support.office.com/article/32f14a23-2c1a-4579-b973-d4b1d78561ad?wt.mc_id=Alchemy_ClientDIA)įrenginyje.

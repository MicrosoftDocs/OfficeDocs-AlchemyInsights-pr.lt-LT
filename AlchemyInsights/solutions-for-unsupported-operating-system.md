# <a name="solutions-for-unsupported-operating-system"></a>Sprendimai dėl "Nepalaikomos operacinės sistemos"



Kai bandote įdiegti "Office 365", galite matyti **nepalaikomos operacinės sistemos** klaidos pranešimą, jei įrenginys neatitinka [sistemos reikalavimų](https://products.office.com/office-system-requirements).
  
"Office 365" reikia:
  
- PC: "Windows 10", Windows 8.1, Windows 8, Windows 7 Service Pack 1, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012 arba Windows Server 2008 R2
    
    ARBA
    
- Mac OS X 10.10
    
Jei šis pranešimas bandant įdiegti "Office" mobiliųjų įrenginių arba "Chromebook", Sužinokite, [kaip įdiegti ir paleisti "Microsoft Office" "Chromebook"](https://support.office.com/article/32f14a23-2c1a-4579-b973-d4b1d78561ad?wt.mc_id=Alchemy_ClientDIA).

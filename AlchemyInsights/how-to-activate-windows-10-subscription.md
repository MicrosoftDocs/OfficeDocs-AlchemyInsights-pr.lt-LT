# <a name="activating-windows-10-subscriptions"></a>"Windows 10" prenumeratų aktyvinimas

Microsoft 365 Business, Microsoft 365 švietimas, Microsoft 365 ir Enterprise teikia skaitmenines licencijas Windows 10 kaip prenumeratos dalis. Norint aktyvinti prenumeratą, jūsų įrenginys jau turi būti suaktyvintas "Windows 10 Pro" (1703 arba naujesnė versija). Jei norite peržiūrėti visą "Windows 10" prenumeratos suaktyvinimo reikalavimų sąrašą, peržiūrėkite ["Windows 10" prenumeratos aktyvinimo reikalavimai](https://docs.microsoft.com/windows/deployment/windows-10-subscription-activation#requirements).

# <a name="sharepoint-publishing-infrastructure"></a>SharePoint publikavimo infrastruktūra


<p>SharePoint yra klasikinis komplektas funkcijų, kurios leidžia autoriui ir skelbti turtingas tinklalapiuose intranete. Aktyvinti SharePoint publikavimo infrastruktūra svetainės rinkinio lygyje juos įjungti.</p>  <p>Daugiau informacijos rasite nuorodoje žemiau.</p>  <ul>  <li><a href="https://support.office.com/en-us/article/Enable-publishing-features-479677A6-8B33-4AC7-907D-071C1C7E4518">Publikavimo priemonių įgalinimas</a></li>  <li><a href="https://support.office.com/en-us/article/Features-enabled-in-a-SharePoint-Online-publishing-site-3AB3810C-3C2C-4361-9D0E-0CBE666EA0B0?wt.mc_id=O365_Portal_MMaven#__toc336865553">Publikavimo priemonių</a></li>  </ul>  <p>Pastabos:</p>  <ul>  <li>Klasikinis publikavimo svetainių šablonai yra leidybos funkcijos pagal numatytuosius nustatymus.</li>  <li>Modernus svetaines klasikinis publikavimo funkcijos negalimos. Modernus svetainės teikti naujesnė negu pagal numatytuosius nustatymus.</li>  </ul>
